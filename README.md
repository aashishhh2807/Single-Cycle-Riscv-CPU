# 32-bit Single Cycle RISC-V CPU (RV32I)

This project implements a 32-bit Single Cycle RISC-V processor using Verilog RTL.

## Features
- Supports RV32I instruction set
- Modular datapath and control design
- Separate instruction and data memory
- Synthesizable RTL design

## Architecture Components
- ALU
- Register File
- Control Unit
- Immediate Generator
- Multiplexers
- Datapath

## Folder Structure

src/        → Top level CPU modules  
components/ → Processor submodules  
programs/   → Assembly and hex programs  

## Tools Used
- ModelSim
- Intel Quartus Prime
